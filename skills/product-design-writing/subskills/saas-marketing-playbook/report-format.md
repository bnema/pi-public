# Report Format

Use this structure for detailed advisory work.

## Executive Summary

- Short overall judgment
- Biggest problem
- Biggest opportunity

## Top Findings

For each finding include:

- Problem
- Why it matters
- Evidence
- Recommendation

Order findings by likely business impact.

## Positioning

- ICP clarity
- pain clarity
- alternative framing
- differentiation
- proof

## Messaging and Copy

- headline and subheadline
- hierarchy
- benefits vs features
- objection handling
- CTA quality

## SEO and Content

- intent match
- topic quality
- structure
- conversion path
- content gap opportunities

## Competitor Notes

- shared patterns
- where the product blends in
- possible whitespace

## Recommended Rewrites

For each rewrite include:

- Original
- Better
- Why this is stronger

## Supporting References

Only include verified sources.

- Direct quote: exact wording with source
- Paraphrase: clearly labeled paraphrase with source

## Experiments / Next Moves

- Quick wins
- Medium-term improvements
- Tests to run

## Open Questions

- Missing context that would change the recommendation
